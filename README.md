# 3D Model Portfolio 🎨

A full-stack 3D model portfolio with Supabase & Vercel.

## Features
- Interactive 3D viewer with Google Model Viewer
- Admin panel to add models
- Dark modern design
- UPI donations (mobile)
- Download models
- Supabase PostgreSQL database
- Deployed on Vercel

## Quick Start

1. Clone this repo
2. Update `backend/.env` with your Supabase credentials
3. Deploy to Vercel
4. Visit admin.html to add models

## Add Models
- Go to /admin.html
- Enter model name and .glb file URL
- Models appear instantly on homepage
